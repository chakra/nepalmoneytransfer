
NEPAL MONEY TRANSFER
============


1) Download nodejs https://nodejs.org/en/ and install.

2) Install IONIC CLI and Cordova
npm install -g ionic cordova 

3) Checkout project 
run => git clone https://github.com/chakra/nepalmoneytransfer.git

4) cd project-folder
then run => ionic serve

Further Reference https://ionicframework.com/docs/v2/setup/installation/

This project has 

=> LoginPage
=> RegistrationPage
=> Tabs(Payee/Transactions)
=> On top of tabs a menu displaying My Details, Payees, Change Password, Change PIN,
Sign Out
